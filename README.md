# Kamenrider_Survivors
Unity 2D Game / 장르 - 로크라이크<br> 
<br> 
Vampire Survivors를 모티브로 제작된 2D Unity 게임<br>
끝없이 몰려오는 몬스터를 처치하며 생존하자!<br>
조작키 - WASD<br>
메뉴키 - ESC<br>
