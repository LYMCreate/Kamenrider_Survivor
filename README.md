# Kamenrider_Survivors
Unity 2D Game / 장르 - 로크라이크

Vampire Survivors를 모티브로 제작된 2D Unity 게임

끝없이 몰려오는 몬스터를 처치하며 생존하자!

조작키 - WASD

메뉴키 - ESC


<구현된 목록>

레벨 업 마다 랜덤으로 얻을 수 있는 10가지의 기술 구현

시간, 레벨에 따라 강해지는 적군 구현

상점으로 스탯 강화를 할 수 있는 기능 구현
